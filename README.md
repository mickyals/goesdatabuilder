# GOES Data Builder
> A repo for building a zarr store for the Advanced Baseline Imager Level 2 Cloud and Moisture Imagery Full Disk ABI-L2-
MCMIPF) product. 




## 📦 Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/mickyals/goesdatabuilder.git
   cd goesdatabuilder
   ```

2. **Create and activate a conda environment**:
   ```bash
   conda create -n goesdatabuilder python=3.9
   conda activate goesdatabuilder
   ```
